# Shaobor Electricity - 电网集成

Home Assistant 自定义集成，用于获取国家电网（95598）的电费和用电量数据。

## 功能特性

- ✅ 实时电费查询（账户余额）
- ✅ 预计可用天数
- ✅ 每日用电量统计
- ✅ 缴费记录查询
- ✅ 用户信息展示
- ✅ 电网标准实体（完全兼容 state_grid_info 格式）
- ✅ 支持年阶梯计费
- ✅ 历史数据存储和查询
- ✅ 多种登录方式（扫码、密码、短信）
- ✅ 自动重连功能

## 安装方法

### 方法 1: HACS（推荐）

1. 在 HACS 中点击右上角的三个点
2. 选择 "自定义存储库"
3. 添加此仓库的 URL：`https://github.com/Shaobor/Shaobor-electricity`
4. 类别选择 "Integration"
5. 搜索 "Shaobor electricity" 并安装

### 方法 2: 手动安装

1. 下载此仓库
2. 将 `custom_components/Shaobor_electricity` 文件夹复制到你的 Home Assistant 配置目录下的 `custom_components` 文件夹中
3. 重启 Home Assistant

## 配置

1. 在 Home Assistant 中进入 **配置** -> **集成**
2. 点击右下角的 **+ 添加集成**
3. 搜索 **Shaobor electricity**
4. 选择登录方式：
   - **扫码登录**（推荐）：使用国家电网 APP 扫码
   - **密码登录**：输入手机号和密码
   - **短信登录**：输入手机号和短信验证码
5. 选择要监控的户号
6. 完成配置

## 传感器说明

集成会创建以下传感器：

| 传感器 | 说明 | 单位 |
|--------|------|------|
| 实时电费 | 账户余额 | CNY |
| 预计可用 | 根据日均消费计算的可用天数 | 天 |
| 最后更新时间 | 数据最后更新时间 | - |
| 缴费记录 | 缴费记录总数 | 条 |
| 用户信息 | 户号、地址等信息 | - |
| 每日电量 | 每日用电量详细数据 | kWh |
| 电网标准实体 | 完整的电费和用电量数据 | 元 |

## 电网标准实体

电网标准实体完全兼容 `state_grid_info` 格式，包含：

- 日均消费
- 剩余天数
- 预付费状态
- 每日用电量列表（daylist）
- 每月用电量列表（monthlist）
- 每年用电量列表（yearlist）
- 计费标准信息（年阶梯计费）

## 阶梯电价配置

⚠️ **重要提示**：默认使用黑龙江省的阶梯电价标准，**不同地区的电价标准不同**！

默认配置（黑龙江省标准）：

- 第1档：0-2040度，0.51元/度
- 第2档：2040-3240度，0.56元/度
- 第3档：3240度以上，0.81元/度

**如何修改为您所在地区的电价：**

目前需要手动修改配置文件。在 Home Assistant 的配置目录中找到 `.storage/core.config_entries` 文件，搜索 `Shaobor_electricity`，修改以下参数：

```json
{
  "ladder_level_1": 2040,    // 第1档上限（度）
  "ladder_level_2": 3240,    // 第2档上限（度）
  "ladder_price_1": 0.51,    // 第1档电价（元/度）
  "ladder_price_2": 0.56,    // 第2档电价（元/度）
  "ladder_price_3": 0.81     // 第3档电价（元/度）
}
```

修改后重启 Home Assistant 即可生效。

**常见地区电价标准参考：**

- **北京**：第1档 0-2880度(0.4883元)，第2档 2880-4800度(0.5383元)，第3档 4800度以上(0.7883元)
- **上海**：第1档 0-3120度(0.617元)，第2档 3120-4800度(0.667元)，第3档 4800度以上(0.917元)
- **广东**：第1档 0-2400度(0.68元)，第2档 2400-3600度(0.73元)，第3档 3600度以上(0.98元)
- **江苏**：第1档 0-2760度(0.5283元)，第2档 2760-4800度(0.5783元)，第3档 4800度以上(0.8283元)

> 注：以上电价仅供参考，请以当地供电公司公布的最新标准为准。

## 自动重连

支持掉线自动重新登录功能（仅密码登录方式）：

1. 配置时启用"自动重连"选项
2. 当登录过期时，会自动使用保存的密码重新登录
3. 无需手动重新配置

## 数据存储

- 历史用电数据会自动保存到本地
- 支持长期历史数据查询
- 数据会定期更新（每10分钟）

## 注意事项

- 首次配置后需要等待几分钟才能获取到完整数据
- 建议使用扫码登录方式，更安全可靠
- 如果遇到登录问题，请删除集成后重新添加
