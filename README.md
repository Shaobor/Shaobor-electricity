# Shaobor Electricity - 电网集成

Home Assistant 自定义集成，用于获取电费和用电量数据。

## 功能特性

- ✅ 实时电费查询（账户余额）
- ✅ 预计可用天数
- ✅ 每日用电量统计
- ✅ 缴费记录查询
- ✅ 用户信息展示
- ✅ 电网标准实体（包含完整的电费和用电量数据）
- ✅ 支持年阶梯计费
- ✅ 历史数据存储和查询
- ✅ 多种登录方式（扫码、密码、短信）
- ✅ 自动重连功能

## 安装方法

### 方法 1: HACS（推荐）

1. 在 HACS 中点击右上角的三个点
2. 选择 "自定义存储库"
3. 添加此仓库的 URL：`https://github.com/Shaobor/Shaobor-electricity`
4. 类别选择 "Integration"
5. 搜索 "Shaobor electricity" 并安装

### 方法 2: 手动安装

1. 下载此仓库
2. 将 `custom_components/Shaobor_electricity` 文件夹复制到你的 Home Assistant 配置目录下的 `custom_components` 文件夹中
3. 重启 Home Assistant

## 配置

1. 在 Home Assistant 中进入 **配置** -> **集成**
2. 点击右下角的 **+ 添加集成**
3. 搜索 **Shaobor electricity**
4. 选择登录方式：
   - **扫码登录**（推荐）：使用 APP 扫码
   - **密码登录**：输入手机号和密码
   - **短信登录**：输入手机号和短信验证码
5. 选择要监控的户号
6. 完成配置

## 传感器说明

集成会创建以下传感器：

| 传感器 | 说明 | 单位 |
|--------|------|------|
| 实时电费 | 账户余额 | CNY |
| 预计可用 | 根据日均消费计算的可用天数 | 天 |
| 最后更新时间 | 数据最后更新时间 | - |
| 缴费记录 | 缴费记录总数 | 条 |
| 用户信息 | 户号、地址等信息 | - |
| 每日电量 | 每日用电量详细数据 | kWh |
| 电网标准实体 | 完整的电费和用电量数据 | 元 |

## 电网标准实体

电网标准实体包含完整的电费和用电量数据：

- 日均消费
- 剩余天数
- 预付费状态
- 每日用电量列表（daylist）
- 每月用电量列表（monthlist）
- 每年用电量列表（yearlist）
- 计费标准信息（年阶梯计费）

## 阶梯电价配置

✨ **自动识别地区**：集成会根据您的电力户号自动识别所在地区，并应用对应的阶梯电价标准！

### 支持的地区

目前支持全国31个省市自治区的电价标准，包括：

- 直辖市：北京、上海、天津、重庆
- 东北：辽宁、吉林、黑龙江
- 华北：河北、山西、内蒙古
- 华东：江苏、浙江、安徽、福建、江西、山东
- 华中：河南、湖北、湖南
- 华南：广东、广西、海南
- 西南：四川、贵州、云南、西藏
- 西北：陕西、甘肃、青海、宁夏、新疆

### 如何工作

1. **自动识别**：集成根据户号前2位数字（地区代码）自动识别您所在的省份
2. **自动应用**：自动应用该地区的阶梯电价标准
3. **可以覆盖**：如果自动识别的电价不准确，您可以手动修改配置文件覆盖

### 查看识别结果

在"用户信息"传感器的属性中，可以看到"识别地区"字段，显示集成识别出的地区名称。

### 手动修改电价（可选）

如果自动识别的电价标准不准确，或您想使用自定义电价，有两种方式修改：

#### 方法1：通过配置界面修改（推荐）

1. 进入 Home Assistant 的 **配置** -> **集成**
2. 找到 **Shaobor electricity** 集成
3. 点击 **配置**（齿轮图标）
4. 在配置界面中修改电价参数
5. 保存后立即生效

#### 方法2：手动编辑配置文件

在 Home Assistant 的配置目录中找到 `.storage/core.config_entries` 文件，搜索 `Shaobor_electricity`，修改以下参数：

```json
{
  "ladder_level_1": 2040,    // 第1档上限（度）
  "ladder_level_2": 3240,    // 第2档上限（度）
  "ladder_price_1": 0.51,    // 第1档电价（元/度）
  "ladder_price_2": 0.56,    // 第2档电价（元/度）
  "ladder_price_3": 0.81     // 第3档电价（元/度）
}
```

修改后重启 Home Assistant 即可生效。

## 自动重连

支持掉线自动重新登录功能（仅密码登录方式）：

1. 配置时启用"自动重连"选项
2. 当登录过期时，会自动使用保存的密码重新登录
3. 无需手动重新配置

## 数据存储

- 历史用电数据会自动保存到本地
- 支持长期历史数据查询
- 数据会定期更新（每10分钟）

## 注意事项

- 首次配置后需要等待几分钟才能获取到完整数据
- 建议使用扫码登录方式，更安全可靠
- 如果遇到登录问题，请删除集成后重新添加
